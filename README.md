# 验证码演示

这是一个简单的验证码演示网页，使用 HTML5 Canvas 动态生成验证码。

## 功能特点

- 随机生成4个字母的验证码
- 字母具有随机的颜色、大小和旋转角度
- 包含干扰线和干扰点
- 验证码不区分大小写
- 支持回车键提交
- 验证成功后显示结果图片

## 在线演示

访问 [https://sdzn2014.github.io/captcha-xfzs](https://sdzn2014.github.io/captcha-xfzs) 查看在线演示。

## 本地运行

1. 克隆仓库
```bash
git clone https://github.com/sdzn2014/captcha-xfzs.git
```

2. 打开 `index.html` 文件即可运行

## 技术栈

- HTML5
- CSS3
- JavaScript
- Canvas API
